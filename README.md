# ScrapLink — Smart Scrap Pickup Platform

## Project Overview

ScrapLink is a simple smart scrap pickup platform developed as a student project. The main aim of this project is to connect households with local kabadiwalas and make the scrap collection process more organized and efficient. In many areas, people generate recyclable waste like newspapers, plastic bottles, and metal items, but they do not have a proper way to schedule pickups. Kabadiwalas also waste time visiting houses when scrap is not ready.

This project provides a basic solution where households can schedule scrap pickups easily, and kabadiwalas can receive requests, manage their routes, and track their work. The system also keeps record of the waste collected, which helps in promoting recycling and maintaining a cleaner environment.

---

## Problem Statement

Many households produce recyclable waste but do not have a reliable system to schedule scrap pickup. Kabadiwalas usually follow fixed routes and often visit homes when scrap is not ready. This leads to time loss, fuel wastage, and inefficient waste collection. There is also no proper tracking system to measure recycling impact.

---

## Proposed Solution

ScrapLink is a platform that allows users to schedule scrap pickup in a simple and convenient way. The system sends pickup requests to kabadiwalas and helps them plan their daily collection more efficiently. It also records the amount of waste collected to support better recycling practices.

---

## Key Features

### For Households

* Schedule scrap pickup
* Select recyclable materials
* Choose pickup date and time
* Receive pickup reminders
* Track recycling contribution

### For Kabadiwalas

* Receive pickup requests
* View daily pickup list
* Manage customers
* Track earnings
* Follow optimized routes

* Mobile / Web Application Interface

---

## How the System Works

1. Household registers in the system
2. User selects scrap materials
3. User schedules pickup time
4. Kabadiwala receives the request
5. Pickup is completed
6. System records the collected waste
